Cell DNA vs. Book Pages comparison

Human Genome Size: The human genome is estimated to contain approximately 3.1 billion base pairs.

  • Wikipedia: "Human Genome"
  • PubMed Central: "On the length, weight and GC content of the human genome" (3,088,269,832 base pairs)

Data Storage Calculation: Assuming 2 bits per base pair (since there are four nucleotides: A, C, T, G), the total size of the human genome can be calculated as follows:

The estimated size is approximately 736.3 MiB.

Page Count Estimation: If we assume an average of 1,000 words per page, with each word averaging 5 letters (plus a space), this results in about 6,000 characters per page. Given that each character is approximately 1 byte, this equates to:

  • 6000 bytes per page
  • $\text{Total pages}=\frac{736.3\text{ MiB}}{6000\text{ B per page}} \approx 1,029,423\text{ pages}$

This means that the human genome's information could fill around 1 million pages of text.
